Getting started with Arduino

This is an "Arduino playground" I put together for a friend. I find it easiest to learn by experimenting with a system that already works, so I wired up a bunch of LEDs, added a few input devices, and wrote some sample code to make it all run.

The 74HC237 chip is a 3-to-8 decoder... it takes a binary number between 0-7 on pins 1-3 and puts 5V on the corresponding output at pins 7,9-15. For example, an input of '101' would put 5V on output #5, pin 10. The datasheet for this chip can be found here.

The 74HC4511 chip is a 7-segment decoder... like the '237, it takes a binary number between 0-7 on pins 2,1,7 and lights up the appropriate segments to display the digit on a 7-segment display. The datasheet can be found here.

The sample projects include:

  • CenterLED: Lights up all LEDs by cycling through them very quickly, but lights up one LED brighter than the others.
  • LEDPot: Uses the analog pot to control the PWM LED brightness and light up one LED in the array.
  • LEDSequence: Cycles through each LED in the sequence.
  • LEDSequenceSwitch: Cycles through each LED in the sequence, reversing direction when the switch is pressed.
  • LEDSequenceSwitchPot: Cycles through each LED in the sequence at a speed controlled by the analog pot, reversing direction when the switch is pressed. Also debounces the switch input so that one press = one direction change, regardless of how long it is held down.
  • Pulse: Pulses the PWM LED on and off.
  • SerialVoltage: Given a digit between 0 and 9 on the serial interface, outputs a corresponding voltage between 0 and 5v on the PWM pin.
  • SerialVoltageDecimal: Given two digits on the serial interface, treats them as a decimal voltage (eg '11' = 1.1v) and puts that voltage on the PWM pin.
